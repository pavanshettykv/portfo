Anti-Theft Helmet system for Dockless Bikes


Synopsis:



Anti-Theft Helmet System provides a great support to a rental bike firm. Usually rental bike firms worried about providing quality bike service for the end users and they failed extremely to monitor the accessories of a rider such as Helmet etc. In this project we have found the solution using IOT to monitor Helmet in real-time.

User Interface: HTML, CSS
Middleware: PHP
Persistent Layer: MySQL

Hardware Interface:
 NodeMCU 12-E,a Wi-Fi module
 Arudino Uno-R3 Microcontroller
 RC522 RFID module
